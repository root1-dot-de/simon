#################################################################################################
# Overall packet structure for NIO implementation:
#

+-----------+-----------+-----------+------------------------------------------------------------
|	index	|	length	|	type	|	description
+-----------+-----------+-----------+------------------------------------------------------------
|	0		|	1		|	Byte	|	packet type
|	1		|	4		|	Integer	|	length of following paket
|	5		|	x		|	n/a		|	packet, see description below
+-----------+-----------+-----------+------------------------------------------------------------

packet descriptions:

+------------------------------------------------------------------------------------------------
|	LOOKUP_PACKET = 0x00		
+-----------+-----------+-----------+------------------------------------------------------------
|	index	|	length	|	type	|	description
+-----------+-----------+-----------+------------------------------------------------------------
|	0		|	4		|	Integer	|	request id
|	5		|	x		|	String	|	name of remote object in lookuptable
+-----------+-----------+-----------+------------------------------------------------------------

+------------------------------------------------------------------------------------------------
|	LOOKUP_RETURN_PACKET = 0x01		
+-----------+-----------+-----------+------------------------------------------------------------
|	index	|	length	|	type	|	description
+-----------+-----------+-----------+------------------------------------------------------------
|	0		|	4		|	Integer	|	request id
|	5		|	x		|	Object	|	requested remote object interface
+-----------+-----------+-----------+------------------------------------------------------------

+------------------------------------------------------------------------------------------------
|	INVOCATION_PACKET = 0x02		
+-----------+-----------+-----------+------------------------------------------------------------
|	index	|	length	|	type	|	description
+-----------+-----------+-----------+------------------------------------------------------------
|	0		|	4		|	Integer	|	request id
|	5		|	x		|	String	|	name of remote object
|	5+x		|	8		|	Long	|	method-hash
|	5+x+8	|	y		|	n/a		|	method parameters: array of primitives and objects
+-----------+-----------+-----------+------------------------------------------------------------

+------------------------------------------------------------------------------------------------
|	INVOCATION_RETURN_PACKET = 0x03		
+-----------+-----------+-----------+------------------------------------------------------------
|	index	|	length	|	type	|	description
+-----------+-----------+-----------+------------------------------------------------------------
|	0		|	4		|	Integer	|	request id
|	5		|	x		|	n/a		|	return value from method invocation: primitive or object
+-----------+-----------+-----------+------------------------------------------------------------

+------------------------------------------------------------------------------------------------
|	TOSTRING_PACKET	= 0x04
+-----------+-----------+-----------+------------------------------------------------------------
|	index	|	length	|	type	|	description
+-----------+-----------+-----------+------------------------------------------------------------
|	0		|	4		|	Integer	|	request id
|	5		|	x		|	String	|	name of remote object in lookuptable
+-----------+-----------+-----------+------------------------------------------------------------

+------------------------------------------------------------------------------------------------
|	TOSTRING_RETURN_PACKET = 0x05	
+-----------+-----------+-----------+------------------------------------------------------------
|	index	|	length	|	type	|	description
+-----------+-----------+-----------+------------------------------------------------------------
|	0		|	4		|	Integer	|	request id
|	5		|	x		|	String	|	String representation of remote object
+-----------+-----------+-----------+------------------------------------------------------------

+------------------------------------------------------------------------------------------------
|	HASHCODE_PACKET	= 0x06	
+-----------+-----------+-----------+------------------------------------------------------------
|	index	|	length	|	type	|	description
+-----------+-----------+-----------+------------------------------------------------------------
|	0		|	4		|	Integer	|	request id
|	5		|	x		|	String	|	name of remote object in lookuptable
+-----------+-----------+-----------+------------------------------------------------------------

+------------------------------------------------------------------------------------------------
|	HASHCODE_RETURN_PACKET = 0x07
+-----------+-----------+-----------+------------------------------------------------------------
|	index	|	length	|	type	|	description
+-----------+-----------+-----------+------------------------------------------------------------
|	0		|	4		|	Integer	|	request id
|	5		|	4		|	Integer	|	hash code of remote object
+-----------+-----------+-----------+------------------------------------------------------------

+------------------------------------------------------------------------------------------------
|	EQUALS_PACKET = 0x08
+-----------+-----------+-----------+------------------------------------------------------------
|	index	|	length	|	type	|	description
+-----------+-----------+-----------+------------------------------------------------------------
|	0		|	4		|	Integer	|	request id
|	5		|	x		|	String	|	name of remote object in lookuptable
|	5+x		|	y		|	Object	|	object to compare with
+-----------+-----------+-----------+------------------------------------------------------------

+------------------------------------------------------------------------------------------------
|	EQUALS_RETURN_PACKET = 0x09
+-----------+-----------+-----------+------------------------------------------------------------
|	index	|	length	|	type	|	description
+-----------+-----------+-----------+------------------------------------------------------------
|	0		|	4		|	Integer	|	request id
|	5		|	1		|	Boolean	|	true or false return value
+-----------+-----------+-----------+------------------------------------------------------------
